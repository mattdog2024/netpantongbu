# 百度网盘定时下载器 (BaiduPan Scheduler)

一款基于 Python 和 PyQt5 开发的 Windows 桌面工具，无需安装百度网盘官方客户端，即可实现**免开发者账号登录**、**文件浏览**和**定时自动下载**功能。

## ✨ 核心特性

- **免申请开发者账号**：通过内嵌浏览器扫码登录，自动提取 Cookie，直接调用网页端接口。
- **定时自动下载**：支持设置下载时间段（如 21:00 ~ 06:00），到点自动开始，超时自动暂停。
- **断点续传**：支持大文件断点续传，随时暂停/恢复。
- **图形化界面**：直观的文件浏览器和任务管理器，操作简单。
- **单文件便携版**：提供打包好的单个 `.exe` 文件，无需安装，放U盘即插即用。

## 🚀 快速开始

### 方式一：直接下载 EXE（推荐）

1. 进入本仓库的 [Releases](https://github.com/mattdog2024/netpantongbu/releases) 页面。
2. 下载最新版本的 `百度网盘定时下载器.exe`。
3. 双击运行即可。

### 方式二：源码运行

1. 确保已安装 Python 3.8+。
2. 克隆本仓库：
   ```bash
   git clone https://github.com/mattdog2024/netpantongbu.git
   cd netpantongbu
   ```
3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
4. 运行程序：
   ```bash
   python main.py
   ```

## 🛠️ 如何自己打包 EXE

本项目已配置 GitHub Actions，每次推送代码到 `main` 分支时，会自动在云端打包并发布到 Releases。

如果你想在本地打包：
1. 双击运行项目目录下的 `build_windows.bat`。
2. 等待几分钟，打包完成后，在 `dist` 目录下会生成 `百度网盘定时下载器.exe`。

## 📝 使用说明

1. **登录**：首次运行点击右上角"登录百度账号"，在弹出的内嵌浏览器中扫码登录。
2. **浏览文件**：左侧面板会显示你网盘中的文件，双击文件夹可进入。
3. **添加任务**：勾选要下载的文件/文件夹，选择本地保存目录，点击"添加到下载队列"。
4. **定时设置**：切换到"定时设置"标签页，勾选"启用定时下载"，设置开始和停止时间，点击保存。
5. **保持运行**：程序会在后台静默等待，到达设定时间后自动开始下载。

## ⚠️ 注意事项

- 本工具基于百度网盘网页端接口开发，下载速度受限于你的账号状态（会员速度快，非会员速度慢）。
- 登录状态（Cookie）保存在程序同目录下的 `bdpan_session.pkl` 文件中，请勿泄露给他人。
- 任务列表保存在 `bdpan_tasks.json` 文件中。
